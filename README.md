# Nexora Digital Solutions 🚀

Nexora Digital is a premium, high-performance digital agency portfolio website. It is designed to showcase world-class services, process workflows, and client results using a modern SaaS-inspired aesthetic.

## 🛠 Technology Stack

### Frontend
- **Framework:** React 18 (Vite)
- **Language:** TypeScript
- **Styling:** Custom Vanilla CSS (SaaS/Dark Mode Theme)
- **Icons:** Lucide React
- **Form Handling:** [Formspree](https://formspree.io/) (Serverless)

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)

### Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/nexora-digital.git
   cd nexora-digital/web-app
   ```

2. **Install & Run Frontend:**
   ```bash
   npm install
   npm run dev
   ```

## ✨ Key Features
- **Premium UI:** Dark mode with violet/indigo glow effects and glassmorphism.
- **3D Animation:** Continuous 3D perspective logo animation.
- **Responsive:** Optimized for Desktop, Tablet, and Mobile.
- **Serverless Contact:** Integrated with Formspree for instant lead generation.
- **Animations:** Subtle micro-interactions and scroll-reveal effects.
